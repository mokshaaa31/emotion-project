# 🎭 Multimodal Emotion Recognition

This project detects human emotions using both video and audio inputs.

## Features
- Video emotion detection
- Audio emotion detection
- Cross-attention fusion

## Run
streamlit run app.py

## Note
Models are downloaded automatically at runtime.

## 🔗 Live Demo
https://emotion-project-1-kbkj.onrender.com

## Features
- Multimodal Emotion Detection (Audio + Video)
- Transformer-based Video Model
- Audio Feature Extraction (MFCC)
- Cross-Attention Fusion

## Tech Stack
- PyTorch
- Streamlit
- OpenCV
- Librosa
